# 📱 천비칠마 상조회 통합 관리 앱

### 1. 프로젝트 개요
* **프로젝트명:** 천비칠마 상조회 통합 관리 시스템
* **목적:** 엑셀(구글 시트)로 관리되던 회비, 지출, 자산 현황을 스마트폰에서 실시간으로 투명하게 확인하고, 회원 개인이 자신의 납부 현황을 조회할 수 있도록 함.
* **개발 기간:** 2026년 1월
* **플랫폼:** Web App (PC/Mobile 반응형)

### 2. 기술 스택 (Tech Stack)
* **Frontend & Backend:** [Streamlit](https://streamlit.io/) (Python 기반 웹 프레임워크)
* **Database:** Google Spreadsheets (실시간 데이터 연동)
* **Hosting:** Streamlit Cloud (GitHub 연동 자동 배포)
* **Version Control:** GitHub

### 3. 주요 구현 기능 (Features)

#### 🏠 홈 화면 (Main)
* **디자인:** 천비칠마 배경 이미지 적용, 다크 모드 기반의 고급스러운 UI.
* **메뉴 구성:** 화면 왼쪽에 3개의 주요 기능을 **달걀형(타원) 대형 버튼**으로 배치하여 직관성 강화.
* **Credit:** 하단에 제작자(GSKim) 표시.

#### 📊 1. 회원 전체 현황 (All Status)
회 전체의 자금 흐름을 투명하게 공개하는 대시보드.
* **탭 1: 분석적 검토**
    * **전체 입금 내역:** 회원별 납부해야 할 금액 vs 납부한 금액 비교 (미납/완납/선납 자동 판별).
    * **회비 통장 지출액:** 조의금, 근조화환, 회의비등 항목별 지출 합계 및 총계 산출.
    * **분석적 검토:** (장부상 잔액)과 (실제 통장 잔액)을 비교하여 차액 검증.
    * **결론:** 차액 발생 원인(이자 수익 등)에 대한 자동 안내 문구 표시.
* **탭 2: 자산 현황**
    * `assets` 시트의 모든 정보(은행, 계좌, 금액 등)를 그대로 출력.
    * 총 자산 합계 자동 계산 (단, 시트 내 '합계' 행은 제외하고 순수 자산만 계산).
* **탭 3: 이자 분석**
    * **적금가입원금:** 장부에서 '적금'으로 지출된 원금 합계.
    * **적금통장가입액:** 현재 적금 통장들의 실제 잔액 합계.
    * **이자발생누적액:** (평가액 - 원금)을 통해 이자 수익 자동 계산 및 강조 표시.
    * **총평:** 자금 관리 투명성에 대한 평가 문구 출력.

#### 🔒 2. 회원 개인 현황 (Personal Status)
개인 프라이버시 보호를 위한 조회 시스템.
* **로그인:** 이메일 아이디(예: `hong`)만 입력하여 간편 로그인.
* **조회 정보:**
    * 본인 성명, 직책, 가입일자.
    * 수령한 혜택(조의 횟수, 조의금 수령액, 근조화환 수령액).
    * **회비 납부 현황:** 총 납부해야 할 금액 대비 실납부액을 계산하여 **미납액/선납액**을 빨간색/파란색 알림으로 표시.

#### 📜 3. 회칙 (Rules)
* **기능:** 회칙 조항을 검색하거나 전체 조회 가능.
* **UI:** `제1조(명칭)` 형태로 제목을 굵게 표시하고 내용은 하단에 배치하여 가독성 확보.

### 4. 데이터 처리 및 로직 (Logic)
* **실시간 연동:** 구글 시트의 데이터가 변경되면 앱에서 '새로고침' 시 약 1분 내에 반영됨.
* **자동 계산:**
    * 날짜 경과에 따른 회비 납부 대상액 자동 계산 (2020년 2월 기산점).
    * 숫자 데이터의 콤마(`,`) 포맷팅 및 문자열 공백 제거 전처리 기능 탑재.
    * 장부의 '입금', '출금' 구분을 인식하여 수입/지출 자동 분류.

### 5. 디자인 시스템 (UI/UX)
* **Dark Mode:** 전체적으로 검은색/어두운 회색 배경을 사용하여 눈의 피로를 줄이고 중후한 느낌 연출.
* **Typography:** 흰색 및 밝은 회색 텍스트로 가독성 최적화.
* **Components:**
    * 입력창 및 결과 박스는 반투명한 어두운 유리 효과 적용.
    * 중요한 숫자(이자, 차액 등)는 녹색/노란색 등으로 색상 강조.
